# 常見錯誤與解法

> 第一次玩 AI 一定會遇到的坑

## 錯誤 1：`ModuleNotFoundError: No module named 'transformers'`

**原因**：你沒裝 transformers 套件。

**解法**：
```
pip install -r requirements.txt
```

如果還是失敗，確認你有先啟用虛擬環境（終端機開頭應該有 `(.venv)`）。

---

## 錯誤 2：`'python' 不是內部或外部命令` (Windows)

**原因**：Python 沒加到 PATH。

**解法**：
重新跑 Python 安裝程式，**勾選**「Add Python to PATH」再安裝。
或者改用 `py` 取代 `python`：
```
py demos/01_sentiment.py
```

---

## 錯誤 3：下載卡住、超慢、或 `ConnectionError`

**原因**：第一次跑要從 HuggingFace 下載模型，網路問題會中斷。

**解法**：
1. 換網路（學校網路通常比家裡 WiFi 快很多）
2. 重新跑程式 ─ HuggingFace 會自動接續下載
3. 設定鏡像（中國/某些地區用）：
   ```
   # Windows PowerShell
   $env:HF_ENDPOINT = "https://hf-mirror.com"

   # Mac/Linux
   export HF_ENDPOINT=https://hf-mirror.com
   ```
   然後重跑 demo。

---

## 錯誤 4：`OSError: [Errno 28] No space left on device`

**原因**：硬碟空間不夠。每個模型 200~500 MB，幾個模型很容易吃掉 2 GB。

**解法**：
- 清硬碟
- 模型快取位置：
  - Windows：`C:\Users\你的名字\.cache\huggingface\`
  - Mac/Linux：`~/.cache/huggingface/`
- 不要用的模型可以直接刪掉 cache 資料夾

---

## 錯誤 5：跑得超慢（一個句子分析要 30 秒）

**原因**：你的電腦沒有 GPU，純 CPU 跑神經網路本來就慢。

**解法**：
- 這個教學的 demo 用 CPU 跑都還 OK，第一次慢，跑第二次會快（模型已經載入記憶體）
- 想快就要 GPU，但這超出新手範圍
- 替代方案：用 Google Colab（免費雲端 GPU）
  - 上 https://colab.research.google.com
  - 新增 notebook，貼程式碼進去
  - 選單 `Runtime > Change runtime type > GPU`

---

## 錯誤 6：`SSLError` 或 `certificate verify failed`

**原因**：公司 / 學校網路有 firewall 擋。

**解法**：
- 換到自己家網路或手機分享網路試試
- 或設定 pip 不檢查憑證（不建議常用）：
  ```
  pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt
  ```

---

## 錯誤 7：Demo 1 跑 OK，Demo 2 報 `OSError: Can't load tokenizer`

**原因**：翻譯模型需要 `sentencepiece` 套件，可能沒裝起來。

**解法**：
```
pip install sentencepiece
```

Windows 上有時會失敗，因為 sentencepiece 需要編譯工具。改裝預編好的版本：
```
pip install sentencepiece --only-binary :all:
```

---

## 錯誤 8：圖片 demo 報 `URLError` 或圖片載不到

**原因**：範例圖網址連不上，或你網路擋住了。

**解法**：
換成別的圖網址，例如：
```python
image_url = "https://picsum.photos/600/400"
```

或下載一張圖到電腦，改用本地路徑：
```python
image_url = r"C:\Users\你的名字\Pictures\test.jpg"
```

---

## 終極除錯心法

1. **把錯誤訊息最後一行** Google 一下 ─ 99% 的錯誤別人都遇過
2. **把錯誤訊息丟給 ChatGPT** 問它怎麼解 ─ AI 解 AI 的錯誤，很自然
3. **重開終端機** ─ 有時候環境變數沒生效，重開就好
4. **重啟電腦** ─ 真的，30% 的問題重啟就好（不是開玩笑）
