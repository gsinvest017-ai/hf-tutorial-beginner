## 立刻可以做的 5 件事（不用學新東西）

### 1. 把 demo 拼成一個小工具
例如：「中文輸入 → 翻成英文 → 做情感分析 → 印出結果」
你已經會這 3 個 pipeline，串起來就好。**這就是你的第一個 AI 應用**。

### 2. 換更厲害的模型
- 情感分析換 `cardiffnlp/twitter-roberta-base-sentiment-latest`（多分類含中性）
- 翻譯換 `facebook/nllb-200-distilled-600M`（支援 200 種語言）
- 圖片辨識換 `microsoft/beit-large-patch16-224`（更準）

去 https://huggingface.co/models 搜尋你的任務，按下載數排序就是熱門模型。

### 3. 試試其他 pipeline 任務
HuggingFace 有 30+ 種 pipeline，你會發現用法都一樣：

| 任務名 | 做什麼 | 試試這個模型 |
|---|---|---|
| `text-generation` | 接龍寫文章 | `gpt2` |
| `summarization` | 摘要長文 | `facebook/bart-large-cnn` |
| `question-answering` | 看文章答問題 | `deepset/roberta-base-squad2` |
| `fill-mask` | 填空題 | `bert-base-uncased` |
| `automatic-speech-recognition` | 語音轉文字 | `openai/whisper-base` |
| `text-to-speech` | 文字轉語音 | `facebook/mms-tts-eng` |
| `zero-shot-classification` | 任意分類 | `facebook/bart-large-mnli` |
| `object-detection` | 物件偵測（框出位置） | `facebook/detr-resnet-50` |

### 4. 做個有畫面的小網站
用 `gradio` 把你的 AI 包成網頁（5 行程式）：
```python
import gradio as gr
from transformers import pipeline

classifier = pipeline("sentiment-analysis")

def analyze(text):
    return classifier(text)[0]

gr.Interface(fn=analyze, inputs="text", outputs="json").launch()
```
跑起來會自動開瀏覽器，你有了一個**真的可以用的 AI 應用**。

### 5. 把成果放上 GitHub
- 開帳號（如果還沒）
- 把你寫的東西 push 上去
- 寫個 README 介紹自己做了什麼
- 履歷可以放連結，這是真的會加分

## 中期可以學的概念

學完 pipeline 之後，下一階段的關鍵字：

### 工具層面
- **Datasets**：HuggingFace 的「資料集 hub」，學會用 `load_dataset()`
- **Tokenizers**：把文字切成 AI 看得懂的單位，深入了解一下會大開眼界
- **AutoModel / AutoTokenizer**：比 pipeline 低一層的 API，給你更多控制權
- **Gradio / Streamlit**：把 AI 包成網頁的工具

### 概念層面
- **Transformer 架構**：所有現代 AI 的底層，看 3Blue1Brown 的影片入門最好
- **Fine-tuning（微調）**：拿別人的模型在你的資料上再訓練一下
- **LoRA**：超省記憶體的微調方法，玩 Stable Diffusion 也會用到
- **RAG（檢索增強生成）**：讓 LLM 「查資料再回答」的技術
- **Embedding**：把文字變數字向量，做相似度搜尋的基礎

## 推薦學習路徑

### 高中生（時間多但基礎少）
1. 把這個教學的 3 個 demo 改一改、玩一玩
2. 上 Coursera 的 [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/)（很多免費）
3. 看 3Blue1Brown 的「Neural Networks」系列影片（中文字幕，神作）
4. 開始用 ChatGPT 當你的「程式家教」，邊問邊寫
5. 試著用 AI 做出一個自己生活會用的小工具（語音記事本、自動翻譯字幕…）

### 大學生（時間少但學習力強）
1. 把 transformers 官方教程跑完：https://huggingface.co/learn/nlp-course
2. 修一門 Python / 機器學習相關課
3. 找一個自己感興趣的領域（金融、生醫、遊戲、藝術…）做專題
4. 把專題放 GitHub，做履歷
5. 找實習，HuggingFace 在台灣有公司用，履歷有 HF 經驗會加很多分
