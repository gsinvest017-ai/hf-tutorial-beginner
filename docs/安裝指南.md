# 安裝指南（一次就好）

> 你只需要做這個一次。之後跑 demo 都不用再裝。

## Step 1：裝 Python

Python 是這個專案使用的程式語言。

### Windows
1. 去 https://www.python.org/downloads/ 下載最新版（**3.10 以上**）
2. **重要**：安裝時第一頁勾選 ✅ **"Add Python to PATH"**（不勾後面會很痛苦）
3. 按 Install Now

### Mac
1. 開啟「終端機」（按 `Cmd + Space` 搜尋 "Terminal"）
2. 貼上這行按 Enter：
   ```
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
3. 裝完後：
   ```
   brew install python
   ```

### 驗證安裝
打開終端機（Windows 按 Win+R 輸入 `cmd`；Mac 開 Terminal），輸入：
```
python --version
```
應該會看到 `Python 3.x.x` 之類的字。**有看到版本號就 OK**。

## Step 2：下載這個專案

### 方法 A：用 git（推薦，之後更新方便）
```
git clone <這個專案的網址>
cd hf-tutorial-beginner
```

### 方法 B：直接下載 zip
1. 在 GitHub 點 `Code` → `Download ZIP`
2. 解壓縮
3. 進到資料夾

## Step 3：建立虛擬環境（強烈建議）

虛擬環境的意思是「**幫這個專案做一個小盒子**」，把套件裝在小盒子裡，
不會弄髒你整台電腦。**強烈建議做**，將來你會感謝自己。

在專案資料夾打開終端機，輸入：

### Windows
```
python -m venv .venv
.venv\Scripts\activate
```

### Mac / Linux
```
python -m venv .venv
source .venv/bin/activate
```

成功的話，終端機最前面會出現 `(.venv)` 之類的字。

## Step 4：裝 HuggingFace 跟相關套件

```
pip install -r requirements.txt
```

這會裝：
- `transformers` ─ HuggingFace 本人
- `torch` ─ AI 模型跑起來需要的引擎（PyTorch）
- `pillow` ─ 處理圖片（給 Demo 3 用）
- `sentencepiece` ─ 翻譯模型需要的小工具
- `notebook` ─ Jupyter Notebook（給 `.ipynb` 版 demo 用）

**會下載大約 2～3 GB**（torch 本身很大），請耐心等。
WiFi 慢的話可能要 10～20 分鐘。

## Step 5：跑你的第一個 Demo

每個 demo 都有兩個版本，**第一次學程式建議用 Jupyter Notebook 版**，可以一格一格慢慢跑、邊跑邊看輸出。

### 方法 A：Jupyter Notebook 版（推薦給初學者）

```
jupyter notebook
```

這會打開你的瀏覽器，左邊會看到資料夾結構。點進 `demos/` → 雙擊 `01_sentiment.ipynb` 打開。

打開之後，**按住 `Shift` 再按 `Enter`** 就會跑一格然後跳到下一格。一路按到底就跑完了。

### 方法 B：純 Python 版（從終端機跑）

```
python demos/01_sentiment.py
```

第一次跑會先下載 AI 模型（大約 250 MB），請耐心等。
**看到結果就代表你成功了。歡迎入坑。**

## 如果中間卡住

→ 看 [`常見錯誤.md`](常見錯誤.md)

最常見的問題：
- `python` 指令不存在 → Step 1 沒勾 "Add to PATH"，重裝一次
- `pip` 裝不動 → 換鏡像：`pip install -r requirements.txt -i https://pypi.org/simple/`
- 下載很慢 → 用學校或公司網路，家裡 WiFi 抓 torch 真的會痛苦

## 完全沒救？

讓會 Python 的朋友或學長姐**陪你裝一次**。
裝環境是 Python 學習中最痛苦的一關，**裝完就一帆風順了**。
